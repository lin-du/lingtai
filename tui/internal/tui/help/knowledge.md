# /knowledge

Browse an agent's private knowledge.

## Aliases

`/knowledge`, `/library`, and `/codex` all open this same view.

## Usage

```
/knowledge
```

## What it does

Browses an agent's private knowledge — local decisions, notes, references, and
migrated codex entries, including its durable knowledge library of accumulated
research and findings. `Ctrl+T` switches between agents.

## When to use it

When you want to read what an agent has recorded and learned for itself, as
opposed to the shared skill catalog (`/skills`) or its system files
(`/system`).
