# /help

Open this help browser.

## Usage

```
/help
```

## What it does

Opens the help browser you are reading now: an overview of how slash commands
work, plus a per-command reference page for every command. Browse the sidebar
with `↑`/`↓`, `Tab` to scroll the content panel, and `Esc` or `q` to return to
the mail view.

## When to use it

Any time you want to look up what a command does or discover commands you
haven't used.
