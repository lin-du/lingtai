# /viz

Open the network visualization.

## Usage

```
/viz
```

## What it does

Opens the agent network visualization in your browser — topology, mail flows,
and agent states. Requires `lingtai-portal` to be on your `PATH`.

## When to use it

When you want a live graphical view of how agents in the network relate and
communicate, rather than the terminal dashboards. For a terminal-only overview
use `/kanban`.
