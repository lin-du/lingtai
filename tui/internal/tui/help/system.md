# /system

Browse an agent's system files.

## Usage

```
/system
```

## What it does

Browses an agent's system files: `system.md`, covenant, principle, procedures,
pad, and llm config. `Ctrl+T` switches between agents.

## When to use it

When you want to read the durable scaffolding that defines an agent — its
governing documents and runtime config — rather than its conversation or
private knowledge (`/knowledge`).
