# /nirvana

Wipe everything and start fresh.

## Usage

```
/nirvana
```

## What it does

Deletes `.lingtai/` and all agent data, returning the project to a completely
fresh state.

## ⚠️ Irreversible

This destroys all agent data and cannot be undone. If you only want a clean
conversation, use `/clear`; to reset while preserving context, use `/molt`.

## When to use it

Only when you genuinely want to start the project over from nothing.
