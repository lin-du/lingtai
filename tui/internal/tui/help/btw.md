# /btw

Ask the agent a side question.

## Usage

```
/btw <your question>
```

## What it does

Delivers your question as an *insight inquiry*. The agent reflects and responds
without interrupting the work it is currently doing — so you can probe its
thinking mid-task without derailing it.

## When to use it

When you want the agent's take on something tangential ("btw, did you consider
X?") and don't want to break its current focus the way a normal message would.
