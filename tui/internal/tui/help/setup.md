# /setup

Run the agent setup wizard.

## Usage

```
/setup
```

## What it does

Opens the agent setup wizard, where you change the LLM provider, model,
capabilities, and runtime settings. Changes propagate to all agents.

## When to use it

For first-time configuration or any substantive change to the agent's provider,
model, or capabilities. For TUI-only preferences (theme, language) use
`/settings`; to create or edit presets, this wizard is where full preset
creation happens.
