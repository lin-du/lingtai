# /skills

Browse the skill catalog.

## Usage

```
/skills
```

## What it does

Opens the skill catalog — the reusable procedures the agent can invoke on
demand. `Ctrl+T` switches between agents; `Enter` on a skill drills into the
files inside that skill's folder.

## When to use it

When you want to see what an agent is capable of doing on demand, or inspect the
contents of a specific skill.
